# web-app-poc
### How to run
* Install npm packages
```
cd web-app-poc
npm install
```
* Build webpack
```
npm run hotbuild
```
* Install bower packages
```
cd ./webapp/assets
bower install
```
* Launch server
```
cd ../../
npm run server
```
* Access through browser
```
http://localhost:3000
```
